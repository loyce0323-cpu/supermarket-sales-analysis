# supermarket-sales-analysis
End-to-end data analysis project using Excel, SQL and R — includes pivot tables, SQL queries and ggplot2 visualizations
# Supermarket Sales Analysis

## Tools Used
- Excel — Pivot tables and dashboard
- SQL — Data queries and analysis
- R — Data visualization with ggplot2

## Charts
- Bar chart — Top 10 customers by revenue
- Line chart — Monthly revenue trend
- Pie chart — Net revenue vs Tax breakdown
- KPI table — Key metrics summary

## Files
- supermarket_sales_dashboard.R — Full R script
- supermarket_dashboard.pdf — All charts combined
